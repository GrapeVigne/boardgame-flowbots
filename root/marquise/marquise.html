Mechanical Marquise 2.0

This is the simplest of the bots and can be used to fill out the player count in dozens of configurations.

Though this bot is straightforward, donâ€™t get complacent. At least one player will need to keep it in check!


Step 1: Gather Warriors:

Form a supply of 25 warriors near you.


Step 2: Place Keep:

Place the keep token in a random corner clearing.


Step 3: Garrison:

Place a warrior in each clearing except diagonally opposite the clearing with the keep token.

Place an extra warrior in the clearing with the keep token.


Step 4: Place Starting Buildings:

Randomly place 1 sawmill, 1 workshop, and 1 recruiter among the clearing with the keep token and those clearings adjacent, with up to one building in each clearing.


Step 5: Fill Buildings Tracks:

Place the remaining 5 sawmills, 5 workshops, and 5 recruiters on the matching Buildings tracks, filling every space except the leftmost space of each track.


Birdsong:

Draw and reveal an order card.

If the order card shows an available item, craft it.


Daylight:

Was the order card a bird card?


Daylight - Battle:

Initiate a battle in each ordered clearing.

  • The defender is the player with the most pieces in the clearing of battle.
  • In the case of a ties, target the player with the most victory points.

Daylight - Recruit:

Place four warriors among ordered clearings the Marquise rule, distributed evenly.

  • If they rule only one clearing, place all four warriors there.
  • If they rule two clearings, place two warriors in each.
  • If they rule three such clearings, place the fourth warrior in such a clearing of highest priority.
  • If they rule four clearings, place one warrior in each.

Daylight - Build:

Place a building in the clearing the Marquise rule with the most Marquise warriors. (The order card determines the type of building placed, not the clearing where it is placed.)

  • If the order card is a fox card, place a sawmill.
  • If the order card is a rabbit card, place a workshop.
  • If the order card is a mouse card, place a recruiter.

Daylight - Move:

Move all but three Marquise warriors from each ordered clearing to the adjacent clearing with the most enemy pieces.


Daylight - Expand:

If a building was placed this turn, or if there are more than five buildings on the map, click "Next".

Otherwise, if the Marquise did not place a building this turn and has five or fewer buildings on the map, do all of the following:

  • Discard the current order card.
  • Then draw and reveal a new order card.
  • Then click "Repeat Daylight" to return to the start of Daylight.

Evening - Score:

Score victory points listed on the rightmost empty space on the ordered Buildings track.

If the order card is a bird card, use the track that would score the most victory points.


Evening - Discard:

Discard the current order card and end the turn.


Daylight - Battle:

Initiate a battle in each clearing.

  • The defender is the player with the most pieces in the clearing of battle.
  • In the case of a tie, target such a player with the most victory points.

Daylight - Recruit:

Place two warriors each in the two clearings the Marquise rule with the lowest priority.

If the Marquise rule only one clearing, place all four warriors there.


Daylight - Build:

Place a building of the type with the most pieces on the map in the clearing the Marquise rule with the most Marquise warriors.

  • If there is a tie between sawmills and any other types, place a sawmill.
  • If there is a tie between workshops and recruiters but not sawmills, place a recruiter.

Daylight - Move:

Move all but three Marquise warriors from each clearing to the adjacent clearing with the most enemy pieces.

Now battle in each clearing just moved into.
